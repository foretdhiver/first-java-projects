이벤트 흐름 : 
1. 메인 페이지의 상단 메뉴 바에 있는 부서별 게시판을 클릭한다.
2. 부서별 게시판  메인 페이지를 보여준다.
	메인 페이지 좌측 부분에는 로그인한 차은우 사원의 사진과 부서/직급이 보이며 그 아래에는 차례대로 공지사항, 관리부게시판, 관리부업무게시판,
	영업지원부게시판, 영업지원부업무게시판, 영업부게시판, 영업부업무지원게시판, 기술지원부게시판,기술지원부 업무게시판,
	프로젝트게시판, 익명게시판이 존재한다
	부서별 게시판 메인 페이지의 바디 부분에는 4개의 구역으로 나눠서 각 부서의 작성일 순으로 최신 게시글 5개의 제목을 보여준다.
	이때 보여주는 게시글을 업무게시글은 제외한다.
	(본인의 부서만 클릭하여 진입 가능하고, 아닐 경우에는 접근 권한이 존재하지 않는다는 alert창이 나온다.)
3. 좌측 부분의 기술지원부 텍스트를 클릭한다.
4. 기술지원부 부서게시판을 보여준다.
	좌측 부분은 부서별 게시판 메인과 동일하며, 부서별 게시판 바디의 상단 좌측 부분에 부서게시판 이라는 텍스트가 있고 우측 부분에는 글쓰기 버튼이 존재한다.
	부서게시판 텍스트 아래에는 글번호, 제목, 작성자(직위), 작성일, 조회수 라는 헤더 아래 헤더 내용에 알맞게 게시글 내용이 작성일 순으로 10건 출력된다.
    이때 보여주는 게시글은 부서 공지사항이 굵은 글씨로 표시되며 게시글의 가장 상단에 위치하게 된다.
    그리고 보여지는 게시글이 오늘 작성되었다면 게시글의 제목 오른쪽에 "NEW" 텍스트가 자동으로 붙어 출력된다
    하단 부분의 좌측에는 검색을 위한 창이 존재하며, 우측에는 다음 페이지로 넘어갈 수 있는 네비게이터가 숫자 텍스로 존재한다.
5. 글쓰기 텍스트를 클릭한다.
6. 글쓰기 팝업창이 보여지며 html 문서형식으로 된 팝업창은 상단에서 하단부 차례대로 부서명,  업무지정 드롭박스, 공지사항 드롭박스, 
         글제목, 텍스트 에디터, 글내용이 존재하며 글제목과 글 내용을 작성할 수 있게 되어 있다.
         부서명은 본인 부서명이 자동으로 표시되며 공지사항 드롭박스는 공란과 부서 공지사항, 사내 공지사항 선택지가 있으며 
         공란이 기본선택이다. 업무지정 드롭박스는 공란과 업무용 선택지가 있으며 공란이 기본 선택이다.
7. 글을 작성하여 팝업창 우측 하단 부분에 글게시 텍스트를 클릭하면 글 게시 여부를 묻는 팝업 창이 보여진다.
           확인 : 글게시를 하려면 확인 버튼을 클릭한다.
           글이 데이터 베이스에 저장되고 글쓰기 팝업창이 닫히고 부서별 게시판 글목록이 보인다.
           취소 : 팝업창만 닫히고 다른 변동사항은 없다.
